# JianYue
简阅一款，清阅读的App应用

介绍 
一款基于自己开发的练手项目。自己封装了lib方便使用，里面使用了一些常用的工具类和自定义控件.主要的功能是通过抓取网页解析成我们想要的页面展示数据.
